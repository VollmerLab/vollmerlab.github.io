            <h3>Coral Resilience Genetics</h3>

            <div style="width: 100%; height: 200px; background: #f8f9fa; border: 2px dashed #ccc; border-radius: 8px; display: flex; align-items: center; justify-content: center; margin: 1rem 0; color: #666; font-size: 0.9rem;">

            <p>We use cutting-edge genomic approaches to understand how corals naturally resist devastating diseases like White Band Disease and adapt to thermal stress. Our genome-wide association studies have identified specific genetic markers that predict disease resistance, enabling precision conservation approaches.</p>

            

            <div class="research-highlights">

                <h4>Key Discoveries:</h4>

                <ul>

                    <li>Disease resistance has a polygenic basis with 10 genomic regions involved</li>

                    <li>As few as 10 genetic variants can predict coral disease resistance with high accuracy</li>

                    <li>Resistant corals use unique immune pathways different from other animals</li>

                    <li>Thermal tolerance genes show population structure across coral ranges</li>

                </ul>

            </div>

            

            <p>This work directly supports coral restoration by helping nurseries select the most resilient coral genotypes for reef rebuilding projects.</p>

        </div>

            <div style="width: 100%; height: 200px; background: #f8f9fa; border: 2px dashed #ccc; border-radius: 8px; display: flex; align-items: center; justify-content: center; margin: 1rem 0; color: #666; font-size: 0.9rem;">

            </div>

            <p>Corals host extraordinarily diverse microbial communities containing both beneficial symbionts and harmful pathogens. We use metagenomics, machine learning, and experimental approaches to understand these complex relationships and identify the key players in coral health and disease.</p>

            

            <div class="research-highlights">

                <h4>Research Approaches:</h4>

                <ul>

                    <li>Machine learning identification of coral pathogens</li>

                    <li>Microbiome manipulation using antibiotics and quorum sensing inhibitors</li>

                    <li>16S rRNA sequencing to profile bacterial communities</li>

                    <li>Network analysis of microRNA-microbe interactions</li>

                </ul>

            </div>

            

            <p>Our work has identified multiple candidate pathogens within bacterial families including Vibrionaceae and Flavobacteriaceae, while revealing the importance of beneficial bacteria in coral health.</p>

        // Enhanced word map interactivity

        document.querySelectorAll('.word').forEach(word => {

            word.addEventListener('mouseenter', () => {

                const text = word.textContent.toLowerCase();

                

                // Define related word groups

                const wordGroups = {

                    'coral': ['acropora', 'cervicornis', 'staghorn', 'pocillopora', 'caribbean'],

                    'disease': ['resistance', 'white band', 'pathogens', 'immunity', 'sctld'],

                    'genomics': ['rna-seq', 'gwas', 'snps', 'transcriptomics', 'machine learning'],

                    'microbiome': ['bacterial', 'vibrio', 'pathobiome', 'quorum sensing', 'antibiotics']

                };

                

                // Highlight related words

                let relatedWords = [];

                for (let [key, group] of Object.entries(wordGroups)) {

                    if (group.includes(text) || key === text) {

                        relatedWords = relatedWords.concat(group).concat([key]);

                    }

                }

                

                document.querySelectorAll('.word').forEach(w => {

                    const wordText = w.textContent.toLowerCase();

                    if (!relatedWords.includes(wordText) && w !== word) {

                        w.style.opacity = '0.3';

                    } else {

                        w.style.opacity = '1';

                    }

                });

            });

            

            word.addEventListener('mouseleave', () => {

                document.querySelectorAll('.word').forEach(w => {

                    w.style.opacity = '1';

                });

            });

        });

    </script>
